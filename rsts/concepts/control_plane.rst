.. _control-plane:

################
Control Plane
################

.. panels::
    :header: text-center
    :column: col-lg-12 p-2

    .. link-button:: divedeep-projects
        :type: ref
        :text: Projects
        :classes: btn-block stretched-link
    ^^^^^^^^^^^^
    **Projects** are a multi-tenancy primitive in Nebula that allow logical grouping of Nebula workflows and tasks, which
    often correspond to source code repositories.

    ---

    .. link-button:: divedeep-domains
        :type: ref
        :text: Domains
        :classes: btn-block stretched-link
    ^^^^^^^^^^^^
    **Domains** enable workflows to be executed in different environments, with separate resource isolation and feature
    configuration.

    ---

    .. link-button:: divedeep-admin
        :type: ref
        :text: NebulaAdmin
        :classes: btn-block stretched-link
    ^^^^^^^^^^^^
    **NebulaAdmin** is the backend that serves the main Nebula API, processing all client requests to the system.

    ---

    .. link-button:: divedeep-console
        :type: ref
        :text: Nebula Console
        :classes: btn-block stretched-link
    ^^^^^^^^^^^^
    **Nebula Console** is the web UI for the Nebula platform.
